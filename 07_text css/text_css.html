텍스트 CSS

font-family = 글꼴

한글과 영문이 함게 작성되는 경우, 영문폰트를 앞에 작성하고, 한글폰트는 뒤에 작성해야 한다. 한글폰트는 영문폰트가 포함된 경우가 많지만, 영문폰트에는 한글폰트가 포함된 경우가 거의 없다.

font-size = 글자의 크기

px, %, em, rem, vw, vh, small, large... 글자크기를 따로 지정하지 않은 경우 기본글자 크기는 16px. 100%는 기본글자크기를 기준으로 했을 때 100%로 적용된다.

font-weight = 글자의 두께

100~900까지의 수치값으로 조절하고, 또는 키워드 lighter(100) / normal(400) / bold(700) / bolder(900) 로 사용 가능. 글꼴마다 지원하는 두께가 다름.

font-style = 글자의 기울임

normal(기본값), italic(기울임)
